## About

Each post lives in its own folder under `posts/` with a `README.md` (the post itself) and an `images/` folder. GitHub renders the markdown in place, so every post has a stable URL you can share or import into Medium via `https://medium.com/p/import`.

Drafts, review notes, and build artifacts live in a separate private repository. This repo holds only the finished work.
